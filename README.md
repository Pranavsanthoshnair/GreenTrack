# GreenTrack - Eco-Awareness Platform

![GreenTrack Logo](./assets/logo.png)

## 🌍 Project Overview

GreenTrack is a comprehensive web platform that promotes environmental sustainability and cleaner communities through technology-driven solutions. Built with React, TypeScript, and Tailwind CSS, it provides users with tools to track eco-friendly habits, connect with their community, and make a real environmental impact.

## ✨ Features

### 🎯 Eco Habit Tracker
- **Daily Habit Checklist**: Track sustainable actions like using reusable bags, biking, water conservation
- **Gamification System**: Earn points, maintain streaks, and level up based on eco-friendly actions
- **Progress Visualization**: Interactive charts and progress bars to monitor improvement
- **Local Storage**: Persistent data storage for habit tracking across sessions

### 🤝 Community Actions Dashboard
- **Event Management**: View and create local environmental events (cleanups, tree planting, workshops)
- **Interactive Map**: Visual representation of community activities and locations
- **Leaderboard**: Community ranking system with badges and achievements
- **Event Categories**: Organized by cleanup, planting, awareness, and recycling initiatives

### 📚 Eco-Awareness Hub
- **Interactive Learning**: Flip cards with sustainability tips and environmental facts
- **Educational Quizzes**: Test knowledge with scoring system and explanations
- **Fun Facts**: Rotating "Did You Know?" section with surprising environmental statistics
- **Category Organization**: Tips organized by energy, water, waste, and transport

### 🛠️ Sustainable Lifestyle Tools
- **Carbon Footprint Calculator**: Calculate weekly CO2 emissions based on lifestyle choices
- **Green Score Assessment**: Personal sustainability rating with improvement suggestions
- **Eco-Product Finder**: Search and filter sustainable products with ratings and eco-scores
- **Daily Reminders**: Customizable notifications for eco-friendly actions

## 🚀 Technology Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom glassmorphism effects
- **UI Components**: shadcn/ui component library
- **Icons**: Lucide React
- **Charts**: Recharts for data visualization
- **Build Tool**: Next.js 14 with App Router
- **Deployment**: Vercel-ready configuration

## 🎨 Design Philosophy

- **Glassmorphism UI**: Modern glass-like effects with backdrop blur
- **Nature-Inspired Palette**: Greens, blues, and earth tones
- **Responsive Design**: Mobile-first approach with seamless desktop experience
- **Accessibility**: WCAG compliant with proper ARIA labels and keyboard navigation
- **Smooth Animations**: CSS transitions and hover effects for engaging UX

## 📱 Responsive Features

- **Mobile Navigation**: Collapsible hamburger menu for mobile devices
- **Adaptive Layouts**: Grid systems that adjust to screen sizes
- **Touch-Friendly**: Optimized button sizes and touch targets
- **Progressive Enhancement**: Core functionality works without JavaScript

## 🌟 Key Highlights

- **Point System**: 10 points per completed habit + streak bonuses

## 🔧 Installation & Setup

### Prerequisites
- Node.js 18+ 
- npm or yarn package manager

### Quick Start
\`\`\`bash
# Clone the repository
git clone https://github.com/yourusername/greentrack.git

# Navigate to project directory
cd greentrack

# Install dependencies
npm install

# Start development server
npm run dev

# Open browser to http://localhost:3000
\`\`\`

### Build for Production
\`\`\`bash
# Create optimized build
npm run build

# Start production server
npm start
\`\`\`
